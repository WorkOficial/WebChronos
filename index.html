Functionality of our advanced detection system

Below you will find our security system that is constantly updated and always improving.

Modular Architecture

The system is based on a central module (AntiCheatCore) that manages penalties, making the code clean and easy to maintain.

Cheat Detection

Includes detection for the most common hacks, such as Speedhack, Fly, Teleport, Super Jump, NoClip, and script injection.

Penalty System

The system applies flexible penalties ranging from kick, mute, temporary ban, and permanent ban.

Monitoring and Alerts

Sends detailed notifications to Discord about detections, allowing developers to monitor cheater activity in real time.

Admin GUI

A hidden in-game administration panel allows admins to manually penalize players, giving them total control over the game.

Intelligent Detection

Uses a suspicion point system to process data asynchronously, which reduces false positives and prevents a player from being banned for a single mistake.

Databases

A Roblox DataStore is used to permanently save bans, ensuring cheaters cannot re-enter the game.

Our technology system in the Cloud

Our Anti-Cheat is powered by a Cloud that allows for better analysis of hackers and maintains performance without causing any lag.

Suspicion Score

The system assigns points to a player each time suspicious behavior is detected. This prevents instant bans and false positives.

Background Processing

More complex detections are not processed in real time but are sent to a simulated "cloud" to be analyzed asynchronously. This reduces server load (CPU and memory) and doesn't interrupt the game.

Deferred Mitigations

Penalties are not applied immediately. The "cloud" only punishes the player when it is certain they are actually cheating, after a more prolonged analysis.
